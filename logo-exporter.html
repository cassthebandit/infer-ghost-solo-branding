How to export: Right-click each logo box → Inspect → select the .export-box element → right-click the node in dev tools → Capture node screenshot (Chrome/Edge) or Screenshot Node (Firefox). This gives you a pixel-perfect PNG with the real DM Sans font. The PNG files included in the package use Liberation Sans as a close approximation.
